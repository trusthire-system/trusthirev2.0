# TrustHire - AI-Powered Hiring Platform

TrustHire is an intelligent hiring platform powered by Machine Learning and OCR (Optical Character Recognition). It streamlines the recruitment process by automatically analyzing and scoring resumes, connecting job seekers with HR professionals efficiently.

## 🚀 Features

### For Job Seekers
- View and search for job postings.
- Create and manage detailed profiles.
- Upload resumes.
- Get automatic resume scoring based on ML algorithms and OCR data extraction.

### For HR Users (Recruiters)
- Post and manage job opportunities.
- Review applications and track automatically segregated applicant pools based on ML-generated scores.
- Approve or reject resume job requests.

### For Admins
- Comprehensive dashboard to manage users (Job Seekers, HR professionals).
- Oversee platform health and system activities.
- Manage system configurations and approve new firm registrations.

## 🧠 Core Technology
- **OCR Engine**: Extracts structured text accurately from uploaded resume documents (PDF, Docx, images).
- **Machine Learning**: Algorithms to evaluate, parse context, and score resumes against job requirements.
