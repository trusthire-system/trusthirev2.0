# TrustHire Frontend

TrustHire's frontend is a high-performance web application built with [Next.js](https://nextjs.org), designed to provide a premium, role-based experience for Candidates, HR professionals, and Administrators.

## 🚀 Key Interfaces

- **Candidate Portal**: Seamless onboarding, automated resume profile builder, and transparent application tracking with skill-gap visualization.
- **HR Dashboard**: Verified organizational environment for posting across-industry vacancies and reviewing AI-shortlisted candidates.
- **Admin Panel**: Centralized control for system configurations and firm registration approvals.

## 🛠️ Tech Stack

- **Framework**: Next.js 14/15 (App Router)
- **Styling**: Vanilla CSS / Tailwind (as per project structure)
- **State Management**: React Hooks and Server Actions
- **Authentication**: JWT-based secure sessions with role-specific guards

## 🚦 Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the results.

## 🏗️ Building for Production

```bash
npm run build
npm start
```

For more details on the overall system architecture, see the [main ARCHITECTURE.md](../ARCHITECTURE.md) in the root directory.
